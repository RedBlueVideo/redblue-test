# \<my-component>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i my-component
```

## Usage
```html
<script type="module">
  import 'my-component/my-component.js';
</script>

<my-component></my-component>
```
